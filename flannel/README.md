# flannel

Install [flannel](https://github.com/coreos/flannel).

## Dependencies

- `curl`
- `upstart`

## Variables

name             | default   | description
-----------------|-----------|----------------------------------
`FLANNEL_VERSION` | `0.6.2` | version to install.
