# tmux

Install [tmux](https://tmux.github.io/).

## Dependencies

- `curl`
- `libevent-dev`
- `libncurses5-dev`

## Variables

name             | default   | description
-----------------|-----------|----------------------------------
`TMUX_VERSION` | `2.3` | version to install.
