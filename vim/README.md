# vim

Install [vim](http://www.vim.org/).

## Dependencies

- `curl`
- `build-essential`
- `libncurses5-dev`

## Variables

name             | default   | description
-----------------|-----------|----------------------------------
`VIM_VERSION` | `8.0.0022` | version to install.
