# stow

Install [stow](https://www.gnu.org/software/stow/).

## Dependencies

- `curl`
- `build-essential`

## Variables

name             | default   | description
-----------------|-----------|----------------------------------
`STOW_VERSION` | `2.2.2` | version to install.
