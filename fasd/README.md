# fasd

Install [fasd](https://github.com/clvv/fasd).

## Dependencies

- `git`
- `build-essential`

## Variables

name             | default   | description
-----------------|-----------|----------------------------------
`FASD_VERSION` | `1.0.1` | version to install.
